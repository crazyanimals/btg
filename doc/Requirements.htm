Hardware Requirements

  • 3D hardware accelerator card 100% compatible with DirectX 9.0c, with full support of Shader Model 1.1 (Shader Model 2.0 support recommended) and latest drivers
  • 1.6 GHz processor
  • 256MB RAM (512MB Recommended)
  • 100% DirectX 9.0c compatible 16-bit sound card and latest drivers
  • Keyboard and one of these controllers: mouse, gamepad, joystick
  • 80MB of uncompressed free hard disk space
  • Network connection (LAN or Internet) for multiplayer game

  • Microsoft Windows XP SP2/XP x64/Vista 32-bit/Vista 64-bit
